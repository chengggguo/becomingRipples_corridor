# Becoming Ripples Corridor Toolbox

## 中文

这个文件夹是 Becoming Ripples 六台装置联动控制更新的工作工具箱。

当前内容：

1. `arduino/190cmBar_device/190cmBar_device.ino`：原始 `190cmBar_code.ino` 的装置端修改版。
2. `arduino/presence_group_controller/presence_group_controller.ino`：每排 `Presence Group Controller` 的 Arduino 程序。

建议从这些文档开始：

- `index.md`
- `docs/development_requirements.md`
- `docs/communication_model.md`
- `docs/wiring_guide.md`

## English

This folder is the working toolbox for the six-device Becoming Ripples control update.

Current contents:

1. `arduino/190cmBar_device/190cmBar_device.ino`: modified device firmware based on the original `190cmBar_code.ino`.
2. `arduino/presence_group_controller/presence_group_controller.ino`: Arduino sketch for each row-level `Presence Group Controller`.

Start from these documents:

- `index.md`
- `docs/development_requirements.md`
